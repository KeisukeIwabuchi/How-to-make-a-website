Block, Element, Modifierの3つでクラスを命名する。
Block ・・・ 塊
Element ・・・ 塊を構成する要素
Modifier ・・・ 要素（または塊）の状態
Component毎に一つのBlockとして命名することで、クラス名の重複を回避できる。
クラス名が長くなりがちだが、SASSなどのCSSメタ言語を使うとスッキリ書ける。
